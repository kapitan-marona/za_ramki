# Офлайн PWA чек-лист замеров

## Что это
Статическое офлайн PWA-приложение (без сервера):
- Шапка: дата (по умолчанию сегодня), адрес, фамилия сотрудника
- Двери: входная + список межкомнатных + заметки
- Окна: список + заметки
- Радиаторы: список + заметки
- Автосохранение локально (localStorage)
- Кнопка "Скачать PDF": генерирует A4 portrait PDF по отдельному шаблону, печатает только заполненные данные

## Запуск локально
Самый простой способ — любой статический сервер.
